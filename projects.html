Gemi Intelligence

A modern web-based chat interface built with HTML, CSS, and JavaScript. The current version features a clean, user-friendly design with image upload capabilities and a responsive layout that works across all devices.

The interface includes smooth animations, interactive elements, and a beautiful dark theme design. Users can upload images and interact with the chat system through an intuitive interface.
